# OpenAI Telegram Bot

## How this is work

1. **Activate virtual environment:**
   ```
   python -m venv venv
   source venv/bin/activate
   ```
2. **Install requirements**
   ```
   pip install -r requirements.txt
   ```
3. **Configure .env**
   ```
   cp .env.sample .env
   nano .env
   ```
4. **Start**
   ```
   python main.py
   ```
